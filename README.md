# Fitness-Flow AI 健身助手

## 应用流程

这是一个完整的健身助手应用，包含以下页面和功能：

### 页面流程

1. **Q1.html** - 基本信息收集
   - 选择年龄范围
   - 选择体重范围
   - 选择性别
   - 完成后自动跳转到 Q2.html

2. **Q2.html** - 性格、目标和经验
   - 选择性格特点（外向/内向等）
   - 选择健身目标（减脂/增肌等）
   - 选择健身经验（初学者/中级/高级）
   - 完成后自动跳转到 Q3.html

3. **Q3.html** - 运动项目选择
   - 选择 1-5 个感兴趣的运动项目
   - 支持篮球、足球、跑步、瑜伽等
   - 完成后自动跳转到 home.html

4. **home.html** - 主页
   - 显示今日进度和训练计划
   - 点击摄像机图标可跳转到 C.html（AR 姿势指导）
   - 底部导航栏可跳转到其他页面

5. **plan.html** - 训练计划详情
   - 查看每周和每月的训练计划
   - 查看各项运动的详细安排

6. **result.html** - 训练结果和进度
   - 查看训练成果
   - 查看身体数据变化
   - 查看成就和奖章

7. **C.html** - AR 摄像头姿势指导
   - 实时姿势检测和反馈
   - 点击关闭按钮返回 home.html

## 导航功能

所有主要页面（home、plan、result）都有底部导航栏：
- **Home** - 返回主页
- **Plan** - 查看训练计划
- **Progress/Results** - 查看训练结果
- **Settings** - 返回 Q1 重新设置

## 数据存储

用户选择的数据会保存在 localStorage 中：
- `q1_age` - 年龄范围
- `q1_weight` - 体重范围
- `q1_gender` - 性别
- `q2_personality` - 性格特点
- `q2_goal` - 健身目标
- `q2_experience` - 健身经验
- `q3_sports` - 选择的运动项目（JSON 格式）

## 如何使用

1. 在浏览器中打开 `Q1.html` 开始使用
2. 按照提示完成所有问卷
3. 系统会自动生成个性化的健身计划
4. 使用底部导航栏在不同页面间切换

## 技术栈

- HTML5
- Tailwind CSS
- Font Awesome Icons
- Plotly.js (图表)
- JavaScript (原生)
