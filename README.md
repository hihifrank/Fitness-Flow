# Fitness-Flow AI 健身助手

## 應用流程

這是一個完整的健身助手應用，包含以下頁面和功能：

### 頁面流程

1. **Q1.html** - 基本資訊收集
   - 選擇年齡範圍
   - 選擇體重範圍
   - 選擇性別
   - 完成後自動跳轉到 Q2.html

2. **Q2.html** - 性格、目標和經驗
   - 選擇性格特點（外向/內向等）
   - 選擇健身目標（減脂/增肌等）
   - 選擇健身經驗（初學者/中級/高級）
   - 完成後自動跳轉到 Q3.html

3. **Q3.html** - 運動項目選擇
   - 選擇 1-5 個感興趣的運動項目
   - 支援籃球、足球、跑步、瑜伽等
   - 完成後自動跳轉到 home.html

4. **home.html** - 主頁
   - 顯示今日進度和訓練計劃
   - 點擊攝影機圖標可跳轉到 C.html（AR 姿勢指導）
   - 底部導航欄可跳轉到其他頁面

5. **plan.html** - 訓練計劃詳情
   - 查看每週和每月的訓練計劃
   - 查看各項運動的詳細安排

6. **result.html** - 訓練結果和進度
   - 查看訓練成果
   - 查看身體數據變化
   - 查看成就和獎章

7. **C.html** - AR 攝影機姿勢指導
   - 即時姿勢檢測和反饋
   - 點擊關閉按鈕返回 home.html

## 導航功能

所有主要頁面（home、plan、result）都有底部導航欄：
- **Home** - 返回主頁
- **Plan** - 查看訓練計劃
- **Progress/Results** - 查看訓練結果
- **Settings** - 返回 Q1 重新設定

## 資料儲存

使用者選擇的資料會儲存在 localStorage 中：
- `q1_age` - 年齡範圍
- `q1_weight` - 體重範圍
- `q1_gender` - 性別
- `q2_personality` - 性格特點
- `q2_goal` - 健身目標
- `q2_experience` - 健身經驗
- `q3_sports` - 選擇的運動項目（JSON 格式）

## 如何使用

1. 在瀏覽器中開啟 `Q1.html` 開始使用
2. 按照提示完成所有問卷
3. 系統會自動生成個人化的健身計劃
4. 使用底部導航欄在不同頁面間切換
